# 🪑 Dube Upholstery Website

A simple React JS website built to showcase the services of **Dube Upholstery**, a small business specializing in furniture and car interior repairs.

## 📌 Project Overview
This website highlights the work and services offered by Dube Upholstery. It is designed to give customers a clear view of what the business does and how to get in touch.

## 🚀 Features
- Home page introducing the business  
- About section describing the organization  
- Services section including:
  - Sofa Repair  
  - Car Trimming  
  - Car Seat Repair  
  - Headboards  
- Clean and simple user interface  
- Responsive design for mobile and desktop  

## 🛠️ Technologies Used
- React JS  
- JavaScript  
- HTML5  
- CSS3

## ⚙️ Installation & Setup
1. Clone the repository:
```
git clone https://github.com/your-username/dube-upholstery.git
```

2. Navigate into the project folder:
```
cd dube-upholstery
```

3. Install dependencies:
```
npm install
```

4. Run the app:
```
npm start
```

The app will run on http://localhost:3000

## 🎯 Purpose
The goal of this project is to help promote a small local business by giving it an online presence where customers can easily view services and learn more about the company.
